# v0.1.0 - First public release

## Highlights

- Single-agent first upgrade path for OpenClaw with `solo`, `duo`, and `team5` profiles
- Bundled `local-memory` plugin `v3.3.0` with layered long-term memory, archive compaction, privacy tiers, and dashboard
- Built-in workflow gates: `/delegate`, `/advisor`, `/verify`
- Safe installer flow with backup, verify, rollback, plugin install records, and generated workspaces
- Shared blackboard templates for task specs, handoffs, advisor review, and QA gates

## Included

- `openclaw-upgrade install --profile solo|duo|team5`
- `openclaw-upgrade verify`
- `openclaw-upgrade rollback --backup <path>`
- Shared workflow skills for delegation, advisor gate, and verification gate
- Claude Code-inspired orchestration patterns rewritten for OpenClaw

## Validation

- Validated on OpenClaw `2026.4.1`
- `npm run test` passed before release
- Recommended default profile: `solo`

## Quick Start

```bash
npx github:Leon-llb/openclaw-upgrade-kit install --profile solo
```
